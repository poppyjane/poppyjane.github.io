我眼中的Mac优势及不足

八月 26, 2017 硬件 本文总阅读量次

用了Macbook Air快一年了，总结一下我认为的它的优势所在及不足。

1.单程序运行，不卡顿，提高工作效率

2.快捷键方便

3.系统自带的支持python，perl等

（1）Mac OS X系统是Unix架构，大部分的软件编译很方便；

（2）惊为天人的设计包括：安装打印机驱动，Mac不需要自己去网站上下载驱动，只需要USB接口连接上电脑，然后根据系统提示点击安装即可，简直棒呆了！

4.轻薄，携带方便，耐操

5.钥匙串访问自动记住密码

现在各种网站应用的账号密码多的数都数不过来，还好有🔑提高效率杠杠的

6.再也不用开关机等候

7.众多优秀的应用，不如ominifocus

8.Mail＋Calendar＋iCloud的组合

Mac与iPhone的协同办公系统让你远离电脑也可以随时随地办公！

9.安装程序

再也不需要自己选择了，没有许多附带应用的自动安装，不需要再清理系统盘。
此外我打开十个以上的应用电脑也不卡，4G的内存，小小容量、海纳百川。

不足之处也很明显

（1）许多软件的不兼容；
（2）惊奇的发现优酷居然连弹幕都木有；
（3）Apple Store上有些应用居然不能退款，首先我不是一个经常申请退款的人，但是有部分软件的恶意收费很恶心，打着别的应用的名义欺骗消费者，需要多改进！
（4）新发布的USB接口取消，如果近两年内U盘没有大改进的话对于部分消费者而言是一种羁绊，本人就很不喜欢电脑周围一大堆线。

未完待续…
