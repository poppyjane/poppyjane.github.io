科研论文的组图方式（1）——PPT组图

2017-07-20
PPT, 科研做图, 组图

厌倦了PPT把图片拖过来拖过去对齐，如何用PPT科学的组图？

演示用到的软件及版本

Microsoft PowerPoint V15.33

  • 位图：jpg，tif，psd，bmp，png；放大有马赛克
  • 矢量图：wmf，emf，eps，cdx（化学反应式)；放大无马赛克
  • 一般提交论文图片：jpg，tif，eps。

Microsoft PowerPoint做简单的汇报（对图片要求较低）

主要运用PPT自带的对齐方式，插入图片——选中需要对齐的图片——图片格式——顶／底部对齐——横／纵向分布——整体调整即可！
  1. 顶部对齐确定顶部的位置
  2. 左对齐确定图片的最左边
  3. 右对齐确定图片的最右边
  4. 纵向分布确定纵向图片间的间距
  5. 横向分布确定横向图片间的间距
  6. 组合上述已经调整好的横纵向分布图片组合为一张大图
